# MUME
라오어의 미국주식 무한매수법을 관리하는 대시보드 MUME 프로젝트
[사이트 바로가기](https://mume.ocko.co.kr/)

### 무한매수법이란?
'무한매수법' 역시 분할매수에 바탕을 둔 투자법이다. 정해진 원금을 쪼개 일정 기간 동안 주식을 매수하여 평균 매수단가를 낮추고, 주가가 평균 매수단가보다 높아졌을 때 수익을 실현하는 것이다.
